DOMtegrity

This project proposes a solution to an unresolved problem: How we can ensure web page is parsed in client's browser free of unauthorized modifications by browser extensions? We designed a protocol utilizing new javascript APIs such as webCrypto and MutationObserver. We carefully considered the disabilities of extensions to establish a secure envirnment free of tampering. We implemented our protocol in an easy-to-install JavaScript framework called DOMtegrity.
